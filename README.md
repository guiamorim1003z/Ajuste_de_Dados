# Ajuste_de_Dados
Projeto de identificação de sistemas aplicando dois métodos de ajuste de sinais: filtragem por funções complexas (DFT + IDFT) e modelo por Mínimos Quadrados (MMQ/ARX). Inclui análise de EMQ, regressão linear e correlação de Pearson para as saídas x(t) e y(t) usando dados reais com ruído.
